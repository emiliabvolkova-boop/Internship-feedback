# Internship Feedback App — CloudLinux

Приложение для сбора фидбека от стажёров и менеджеров.

## Что внутри
- **Главная** — выбор роли (стажёр / менеджер / админ)
- **Форма стажёра** — self-assessment с core + milestone вопросами
- **Форма менеджера** — оценка стажёра с core + milestone вопросами
- **Админ-панель** — дашборд со всеми ответами (пароль: `feedback2026`)

---

## Деплой на Vercel — пошаговая инструкция

### 1. Создай GitHub аккаунт (если нет)
Зайди на https://github.com и зарегистрируйся.

### 2. Создай новый репозиторий
- Нажми зелёную кнопку **"New"** (или зайди на https://github.com/new)
- Имя: `internship-feedback` (или любое другое)
- Оставь **Public** (или Private — без разницы)
- **НЕ** ставь галочку "Add a README file"
- Нажми **Create repository**

### 3. Загрузи файлы
На странице нового репозитория нажми **"uploading an existing file"** и перетащи ВСЕ файлы из этой папки:
```
feedback-app/
├── index.html
├── package.json
├── vite.config.js
└── src/
    ├── main.jsx
    └── App.jsx
```
Нажми **Commit changes**.

### 4. Подключи Vercel
- Зайди на https://vercel.com
- Нажми **Sign Up** → **Continue with GitHub**
- Разреши доступ к репозиториям
- На главной нажми **"Add New..." → "Project"**
- Найди репозиторий `internship-feedback` и нажми **Import**
- Настройки оставь по умолчанию (Vercel сам определит Vite)
- Нажми **Deploy**

### 5. Готово!
Через ~1 минуту получишь ссылку вида:
```
https://internship-feedback-xxx.vercel.app
```
Эту ссылку можно отправить стажёрам и менеджерам!

## Важно знать

⚠️ **Сейчас данные хранятся в localStorage браузера** — это значит:
- Каждый пользователь видит только свои ответы
- Админ-панель показывает данные только из того же браузера

Для полноценной работы (все ответы в одном месте) нужно подключить базу данных — это следующий шаг. Варианты:
- **Google Sheets** (через Apps Script) — бесплатно, просто
- **Firebase** — бесплатно, реал-тайм
- **Supabase** — бесплатно, PostgreSQL

Напиши мне в Claude, и мы подключим хранение данных!
